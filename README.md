<h1>개발목적</h1>
다수의 기업, 개인을 대상으로 하는 크라우드펀드를 작성, 관리하는 홈페이지를 작성한다.<br/>
회원관리, 상품등록, 구매 및 결제 등 기본적인 쇼핑몰의 기능을 모두 포함하며 후원자와 창작자가 의사소통을 할수 있는 게시판 기능을 강화한다.
